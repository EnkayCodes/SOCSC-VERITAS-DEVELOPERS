# SOCSC VERITAS DEVELOPERS — Official Repository

Welcome to the **SOCSC VERITAS DEVELOPERS** official GitHub repository!   

This is the central hub for all **tutorial guides**, **code examples**, **learning resources**, and **practical exercises** designed to support your journey into blockchain development, especially with **Sui Move**, **Web3**, and supporting technologies.

Whether you're a beginner exploring your first smart contract or an advanced builder working on full dApps, this repo is your structured learning companion.

---

##  What You’ll Find Here

###  Tutorial Guides
Step-by-step explanations of important Sui Move concepts, system modules, syntax, and real-world usage.

###  Code Examples
Well-commented code snippets and full project samples to help you understand and build faster.

###  Practice Tasks & Challenges
Mini projects and challenges after each tutorial to test your knowledge.

###  Example Packages
Complete Move packages you can explore, clone, and modify.
